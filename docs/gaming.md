# Installing Gaming Tools on Arch-Based Systems with Pamac (GUI Method)

This guide will walk you through installing **Bottles**, **ProtonUp-Qt**, **Lutris**, **Steam**, and **Heroic Games Launcher** using the **Pamac GUI** on Arch-based Linux distributions.

---

## 1. Enable AUR Support in Pamac

Some applications (like `protonup-qt` and `heroic-games-launcher-bin`) are available in the **AUR (Arch User Repository)**.  
You need to enable AUR support before installing them.

1. Open **Pamac** (look for "Add/Remove Software" in your applications menu).
2. Click the **☰** menu in the top right corner.
3. Select **Preferences** (you may be asked for your password).
4. Go to the **Third Party** tab.
5. **Enable AUR support** and optionally check **Enable AUR updates**.
6. Close the Preferences window.

---

## 2. Install the Applications

1. In the Pamac search bar, type **bottles** and install it.
2. Search for **protonup-qt** and install it.
3. Search for **lutris** and install it.
4. Search for **steam** and install it.
5. Search for **heroic-games-launcher-bin** and install it.

💡 You may need to scroll down or check the **AUR** tab in the search results to find some packages.

---

## 3. After Installation

- **Steam** will install its required runtime on first launch.
- **ProtonUp-Qt** can download and manage custom Proton-GE versions for Steam and Lutris.
- **Bottles** helps you run Windows apps with Wine in isolated environments.
- **Lutris** and **Heroic Games Launcher** make managing games from multiple sources easy.

---

**You’re ready to game on Linux!** 🎮
