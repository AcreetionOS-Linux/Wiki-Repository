# Contributing to AcreetionOS

We welcome contributions from everyone. Whether you are a developer, a designer, or a user who wants to help improve the documentation, you can make a valuable contribution to AcreetionOS.

This document provides templates for various types of contributions. Please use them to ensure that your contributions are easy to understand and process.

## Build Information

To help us diagnose and resolve issues more effectively, AcreetionOS generates a build information file on your machine. This file contains details about your specific build, including the build number. We may ask for this information when you report a bug or seek support. It is generated at build time and helps us understand the exact state of your system, allowing us to provide you with the best possible assistance.
